## 使用约束

- 当前文件是视觉草案，不是最终发行资产。
- 正式进游戏前，需要逐张检查比例、裁切、清晰度和授权边界。
- 切片来自接触表，部分图像可能包含边缘留白或相邻格干扰，后续应替换为单张重绘图。
- 正式资源应保持统一风格：高一点的圆胖梨形、哑光手绘黏土质感、肚肚作为视觉中心，避免过度光滑的 3D 渲染感。
